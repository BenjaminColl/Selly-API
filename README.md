# Selly API

This is an API to be used with Selly (http://selly.gg).

The features include the following:

- Get all coupons
- Get Specific coupon
- Create Coupons
- Update coupons
- Get all orders
- Get specific order
- Get orders by page number
- Get all products
- Get specific products
- Update products
- Delete products
- Get all groups
- Get specific groups
- Get all messages
- Get specific messages
